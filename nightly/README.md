# Aurora — Mejora Nocturna (Inicio)

Sistema de mejora continua nocturna del sistema de diseño Ntizar Aurora.

## Cómo funciona

Cada noche, 4 iteraciones automáticas:

1. **01:00 UTC** — Investigación web de tendencias y mejores prácticas
2. **02:00 UTC** — Análisis de gaps + aplicación de Mejora CSS #1
3. **03:00 UTC** — Aplicación de Mejora CSS #2
4. **04:00 UTC** — Aplicación de Mejora CSS #3 + Reaprendizaje

## Logs

- `research-log.md` — Hallazgos de investigación web
- `iteration-log.md` — Historial de mejoras aplicadas
- `summary.md` — Resumen nocturno
